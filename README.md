# HomiesCar 汉米资料档案库

HomiesCar 汉米，简称汉米会，是若尘于 2024 年 6 月在武汉创办的小米汽车车友会。

截至当前记录，汉米会会员规模已超过 6000 人。过去两年来，汉米会持续组织车友活动，覆盖节日聚会、露营、跑山、嘉年华、滑雪、后备箱集市、年会、婚车服务等场景，并获得车友广泛好评。

这个项目用于记录汉米会的一切：组织信息、活动档案、运营资料、影像素材索引、会员与车标资料、平台内容线索，以及未来可以继续沉淀的故事和方法。
