# HomiesCar 汉米会官网设计

## 目标

创建一个可直接打开的静态单页网站，对外介绍 HomiesCar 汉米武汉小米车友会，展示社群规模、活动气质、平台账号和加入/合作方式。

## 内容来源

- `docs/profile.md`：组织介绍、创办时间、会员规模。
- `docs/timeline.md`：活动时间线和代表活动。
- `docs/platforms.md`：抖音、小红书、小米汽车 APP 社区信息。
- `assets/platforms/`：平台主页截图。
- `assets/site/`：从 `D:\HomiesCar汉米` 筛选并压缩的真实活动照片。

## 页面结构

1. Hero：真实跑山车队航拍作为第一视觉，展示品牌名、城市、创办时间、6000+ 会员和行动按钮。
2. 数据带：展示会员规模、活动记录、平台数据、城市关键词。
3. 车友会介绍：用短文案说明汉米会定位和活动风格。
4. 活动精选：展示代表活动时间线和照片卡片，包括嘉年华、露营、跑山、滑雪、年会、后备箱集市等。
5. 社群场景：露营、烧烤、自驾、跑山、婚车、观影、电竞、滑雪等。
6. 平台矩阵：抖音、小红书、小米汽车 APP 社区，并展示已归档的主页截图。
7. 加入与合作：加入请私信，商业合作、活动举办可私。

## 视觉方向

高级城市夜跑感：深色金属底、青绿色电光、细线网格、真实车队照片、克制的动效和玻璃质感面板。整体避免普通企业官网的白底卡片感，突出武汉本地车友会的速度、秩序、温度和社群能量。

## 技术方案

- `index.html`：单页结构和内容。
- `styles.css`：响应式视觉系统、布局、动效。
- `script.js`：导航滚动、入场动画、计数轻动效。
- 不依赖构建工具，不需要服务器，直接打开 `index.html` 可用。

## 验证

- 检查桌面与移动宽度下无文字重叠。
- 检查所有图片路径有效。
- 用浏览器打开 `index.html` 做视觉确认。
